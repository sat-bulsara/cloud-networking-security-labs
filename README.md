# Azure Cloud Engineering Labs

Hands-on Azure labs focused on networking, identity, security, and infrastructure fundamentals.

## Key Areas Covered

- Virtual Networks & Subnets
- Network Security Groups (NSGs)
- Identity & RBAC (Entra ID)
- Storage & Access Control
- Resource Dependencies
- Cost Awareness & Optimisation

## Approach

All labs are:

- Built using Azure CLI (not portal-first)
- Tested through failure scenarios
- Documented with real troubleshooting steps
- Designed to simulate real-world cloud engineering tasks

## Example Labs

- Lab 03 — First Azure VM (network + compute)
- Lab 05 — Resource Dependencies (real Azure behavior)
- Lab 07 — NSG Firewall Rules
- Lab 09 — Identity & Access Control (RBAC)

## Outcome

These labs demonstrate:

- Ability to build cloud infrastructure
- Understanding of networking & security layers
- Troubleshooting and debugging skills
- Real-world Azure architecture thinking
